Grid Container

Para fazer um elemento como grid Container, é so definir como: grid ou inline-grid 

 Property:
            grid-tamplate-columns = definir o numero de colunas com a largua deles 
            grid-template-rows    = definir a altura de cada um (linha)
            justity-content       = usado para alinhar todos os grids dentro do container
                space-evenly (espaço uniforme tds), space-around (espaço ao redor), space-between (espaço entre), 
                center, start, end
            align-content         = Usado para o alinhamento verticalmente para cada comp dentro do conteudo
                msm proproedade de cima
