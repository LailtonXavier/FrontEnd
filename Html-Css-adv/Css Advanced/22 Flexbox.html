Flexbox

Para usar um flexbox, voce precisa definir um flex container
Os intems dentro seria nosso Flex items 

        Properties:
            flex-direction: define a direção > items > column (reverse), row (reverse)
            flex-wrap: agrupados ou n > wrap (quebra se for possivel), norap (n quebra, deixa na msm linha)
            flex-flow: abreviação de flex-direction e flex-wrap > row wrap (pega os dois)
            justify-content: alinhar itens > center, flex-start (no começo), flex-end (fim), space-around (espaço ao redor), space-between (espaço entre)
            align-items: alinha no meio > center; height: 200px, flex-start (top), flex-end (bottom), stretch (estica ate preencher), baseline (alinha msm um maior que o outro)
            align-content: alinhar as linhas > space-between (espaço linha), space-around (ao redor), stretch (padrao), center, flex-start, flex-end
