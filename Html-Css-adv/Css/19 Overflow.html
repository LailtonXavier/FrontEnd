Overflow

Quando o conteudo é muito grande, ai entra ele.

Vc pode usar quando quer ter mais controle sobre seu layout

        Propriedades:
        visible     =   padrão, n é cortado, o conteudo fica fora da caixa
        hidden      =   cortado, o resto do conteudo fica invisivel
        scroll      =   cortado porem a barra de rolagem é acrescentado
        auto        =   semelhante ao scroll, mas adiciona barra qnd necessario
        servi tbm para navegação, ou seja, os links n iram quebrar
Overflow so funciona para elementos de bloco com altura e largura

overflow-x: hidden = esconde scrol horizontal
overflow-y: scroll = adiciona vertical scrol
