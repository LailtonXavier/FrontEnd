Margin

Usada para criar espaço ao redor dos elementos,
fora de qualquer bordas definidas, são elas:
Superior, direita, inferior e esquerda

Lados individuais

margin=right, margin-bottom e margin-left
comprimento: px, pt, cm, mm. etc.

Shorthand property abreviação

A sua abreviação fica assim

        Propriedades:
        3 valores = (Superior) + (direita e esquerda) + (inferior)
        2 valores = (Superior e inferior) + (direita e esquerda)
        1 valor = todos os quatros margens tem o msm tamanho
        auto =     centralizar horizontal e dividido nas larguras

Inherit value valor herdado

Este elemento vai ser herdado do pai com inherit
