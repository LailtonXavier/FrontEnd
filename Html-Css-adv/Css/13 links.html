Links

Um link pode ter varios estilos e estados diferentes 

        Essa é a regra da ordem:
        a:link    - um link normal n visitado
        a:visited - um link que o usuario visitou
        a:hover   - um link qnd o usuario passa o mouse sobre ele
        a:active  - um link no momento em é clicado
