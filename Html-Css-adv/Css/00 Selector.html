Seletor simple com nome, ID, classe

Seletor por Nome

"P" é um seletor que aponta para o html
"color" é uma propriedade e
"lawngreen" é um valor da propriedade

Seletor ID

O seletor ID usa atributo de identificação de um elemento Html
ele é unico
no css é usado um "#"

Seletor Classe

ele tambem seleciona elementos do Html
no Css é puchado por um (.) ponto seguindo o nome da classe

expecificar apenas um elemento

Resumo

            Todos os seletores simples de CSS
    Selector	    Example     Example description
    .class	    .intro	Selects all elements with class="intro"
    #id	            #firstname	Selects the element with id="firstname"
    *	            *	        Selects all elements
    element	        p	Selects all [p] elements
    element,element,..	div, p	Selects all [div] elements and all [p] elements
