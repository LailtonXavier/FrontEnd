Padding

Usado para criar um espaço ao redor de qualquer > elemento dentro de sua borda
possui valor: superior, direito, inferior e esquerdo

        Padding top
        Padding right
        Padding botton
        Padding left

        comprimento -   especifica um preenchimento em px, pt, cm, etc.
        %           -   especifica um preenchimento em% da largura do elemento que o contém
        herdar      -   especifica que o preenchimento deve ser herdado do elemento pai
        (Valores negativos n são permitidos)

        Propriedades de Valor:
        preenchimento = 1   todos os lados
        preenchimento = 2   (superior e inferior) + (direito e esquerdo)
        preenchimento = 3   top, R and L, e bottom
