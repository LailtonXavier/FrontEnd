Calculando a altura e largura de um elemento

Para defenir corretamento a largura e altura de um elemento, é preciso adicionar padding, border e margin

Calculando Largura total de um elemento:
Largura total do elemento = width + left padding + right padding + left border + right border + left margin + right mergin

Calculando altura total de um elemento:
Altura total do elemento = height + top padding + bottom padding + top border + bottom border + top margin + bottom margin
