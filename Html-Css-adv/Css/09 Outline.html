Contorno/Outline

Contorno é uma linha desenhada em torno dos elementos, FORA das bordas, para fazer o elemento se destacar 

        As propriedades de contorno:
            outline-style 
            outline-color 
            outline-width 
            outline-offset 
            outline

Outline style: dotted - dashed - solid - double - groove - ridge - inset - outset - none - hidden

Width Outline/Largura do contorno, são os mesmo da border

Outline Color/Cor do contorno: tem um outline-color: invert = realiza a inversão de cores independente da cor do fundo

Outline Shorthand/ abreviação

For Example:
outline: 4px ridge pink

Outline Offset

Propriedade que adiciona espaço entre o contorno e a borda
