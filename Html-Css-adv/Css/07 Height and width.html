Altura e largura

Ela define a altura ou largura dentro do preenchimento, dorda e margem do elemento 

        Valores:
        auto        o navegador calcula altura e largura
        length      define em px, cm, etc
        %           porcentagem que o bloco contem
        initial     altura ou largura de seu valor padrao
        inherit     Herdada de seu valor pai
