Layout de um site

Geralmente os sites exibem conteudos em varias colunas
(como revista ou jornal

        header   Define um cabeçalho para o documento ou seção
        nav      Define um conjunto de links de navegação 
        section  Define uma seção em um documento
        article  Define o conteudo da pagina
        aside    Define o conteudo na barra lateral
        footer   Definir um rodape para um documento ou seção 
        details  Definir detalhes adicionais que o usuario pode abrir e fecha sob chamada
        summary  Defina um titulo para o elemento details
