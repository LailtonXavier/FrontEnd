Elemento principal

Head é um recipiente para os elementos:
title, style, meta, link, script e base

O head é um conteiner para metadados, os metadados sao dados
dados sobre o documento Html, definem titulo do documento, conjunto de caracteres
estilos, scripts e outras informaçoes meta. 

            Elemento title: titulo do documento 
            Elemento style: define o estilo da pagina
            Elemento link: define relação entre o ducumento atual e externo

            Elemento meta: especifica conjunto de caracteres, descrição 
              da pagina, palabras-chave, autor, configuração da janela. 
                Conjunto de caracteres: meta charset="UTF-8"
                Palavras-chave p/ mecanismo de pesquisa: meta name="keywords" 
                    content="HTML, CSS, JavaScript"
                Descrição web: meta name="description" content="Free web tutorial"
                Definir autor de uma pagina: mata name="author" content="John"
                Atualizar o documento: meta http-equiv="refresh" content="30"
                Definir a janela visivel para tds os dispositivos: meta name="viewport"
                    content="width=device-width, initial-scale=1.0"

Script define o JavaScript do lado do cliente

Base especifica o URL base e/ou destino para tds Urls relatvos
especifica um modelo padrao para o site

                    Resumo do capitulo

                        O head elemento é um contêiner para metadados (dados sobre dados)
                        O head elemento é colocado entre a html tag e a body tag
                        O title elemento é obrigatório e define o título do documento
                        O style elemento é usado para definir informações de estilo para um único documento
                        A link tag é mais frequentemente usada para vincular a folhas de estilo externas
                        O meta elemento é normalmente usado para especificar o conjunto de caracteres, 
                                descrição da página, palavras-chave, autor do documento e
                                 configurações da janela de visualização
                        O script elemento é usado para definir JavaScripts do lado do cliente
                        O base elemento especifica o URL base e / ou destino para todos os URLs relativos em uma página
